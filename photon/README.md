# Project Photon

Babylon.jsで軽量かつ操作が容易な3Dエディタを実装しました。


## 使い方 (一般)
1. デプロイ先のエディタにアクセス [リンク](https://photon-seven.vercel.app/editor)


## 使い方 (開発)
1. このリポジトリをクローンまたはフォークする
2. `yarn`
3. `yarn dev`
5. http://localhost:3000/ にアクセスする


もしくは
1. https://photon-tau.vercel.app/ にアクセスする
2. 「開発用ページ(editor)」をクリック
