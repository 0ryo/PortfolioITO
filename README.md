## プロジェクト一覧

### 1. Photon  
昨年度の卒業研究で開発した、軽量な Web ベースの 3D モデルビューワーです。  
ブラウザ上で動作し、さまざまな 3D フォーマットの読み込み・表示・操作が可能です。

### 2. PocketGarden  
Swift を用いて個人制作した ToDo アプリケーションです。  
基本的なタスク管理機能に加え、25 分集中タイマー、タスク達成によるプレイヤーレベルアップエフェクトを備えています。
